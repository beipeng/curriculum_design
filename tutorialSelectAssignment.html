TUTORIAL

A library of assignments will be provided in a 4x4 grid world. You can select any assignment in any order to design your curriculum.

To add an assignment to the curriculum, you should

1. select a room layout by clicking the image (after clicking it, a larger image will be shown to help you identify it)
           

2. select a corresponding command to train the dog to execute in this selected room layout
                                             

3. click to add the assignment to the curriculum

Now, please try adding an assignment to the curriculum!
